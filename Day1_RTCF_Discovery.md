# Day 1: RTCF Framework Discovery
**Date:** January 1, 2026

## What I Learned
The RTCF Framework (Role + Task + Context + Format) is the core skill of prompt engineering.

~~~ The Formula
  ROLE: Act as [profession/role]
  TASK: [Action verb] [what to do]
  CONTEXT: [Background info]
  FORMAT: [How to structure output] 
  ~~~

**Why it works (LLM perspective):**
- LLMs **copy patterns** better than follow rules
- **Examples > instructions**
- **Last strong signal > first instruction**
- **Roles define WHO, examples define HOW**
